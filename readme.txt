Pada tes berikut ini saya menggunakan Helper untuk fungsi kelipatannya
pada variable $input adalah inputan nilai dari user untuk dilakukan proses

pertama akan dilakukan foreach sejumlah nilai inputan
jika nilai input modulus 3 dan modulus 5 tidak sama dengan 0
maka akan dilakukan pencarian sampai inputan tersebut %3 maka akan dicetak kata "marlin"
dan ketiak % 5 akan dicetak kata "booking"
dan ada kondisi dimana jika modulus 3 dan 5 maka akan mencetak "marlinBooking"

setelah itu ada kondisi lagi untuk mengecek apakah inputan bisa dibagi3 dan 5 jika iya maka,
yang bisa dibagi 3 akan dicetak booking
jika bisa dibagi 5 akan dicetak marlin

jika keduanya maka akan dicetak "Booking Marlin"
